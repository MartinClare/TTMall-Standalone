METHOD 1: Deploy via GitHub (Recommended)
==========================================

Step 1: Push to GitHub
----------------------
git init
git add .
git commit -m "Deploy to Railway"
git remote add origin https://github.com/YOUR_USERNAME/TTMall-Standalone.git
git branch -M main
git push -u origin main

Step 2: Deploy on Railway Website
----------------------------------
1. Visit: https://railway.app
2. Click "Start a New Project"
3. Choose "Deploy from GitHub repo"
4. Select: TTMall-Standalone
5. Wait for deployment (2-3 minutes)

Step 3: Add Volume (IMPORTANT!)
--------------------------------
1. Click your service in Railway
2. Go to: Settings ‚Üí Volumes
3. Click: + New Volume
4. Mount path: /app/public/uploads
5. Size: 1GB (or more)
6. Click: Add

Step 4: Get Your URL
--------------------
1. Go to: Settings ‚Üí Domains
2. Click: Generate Domain
3. Copy your URL (e.g., https://ttmall-production.up.railway.app)
4. Visit and test! üéâ


METHOD 2: Deploy via Railway CLI
=================================

Step 1: Install Railway CLI
----------------------------
npm install -g @railway/cli

Step 2: Login to Railway
-------------------------
railway login

Step 3: Initialize & Deploy
----------------------------
railway init
railway up

Step 4: Open Dashboard to Add Volume
-------------------------------------
railway open

Then follow Method 1, Step 3 above to add volume.

Step 5: Get Deployment URL
---------------------------
railway domain
